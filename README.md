# smart-extension-esp32c3
ESP32-based smart extension with WiFi control, safety, and energy monitoring

FEATURES INCLUDED:
3 Relays – Remote ON/OFF (Blynk)
Load detection before switching (via ACS712)
Over-current cut-off (safety)
Temperature-based protection (NTC / LM35)
Energy monitoring (kWh – basic calculation)
Blynk notifications + Alert section
Live current flow indicator (LED widget)
